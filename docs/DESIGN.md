# DESIGN.md

## Design Vision

Masakanku harus terasa seperti:
- personal
- hangat
- calming
- cute but clean
- feminine without looking childish
- modern AI assistant, bukan aplikasi resep tradisional

Mood utama:
> “cozy smart kitchen companion”

UI bukan sekadar pink everywhere. Fokusnya adalah soft emotional design dengan nuansa:
- warm
- homey
- playful
- elegant
- comforting

Inspirasi visual:
- Pinterest cooking aesthetic
- Little corner cafe vibe
- Korean lifestyle apps
- Lemon8 / Meitu soft UI
- modern journaling apps
- pastel productivity apps
